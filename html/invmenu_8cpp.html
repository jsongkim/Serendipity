Displays and manages the inventory menu.

Parameters
arrayArray of bookType objects representing the inventory
bookCountCurrent number of books in the inventory
Returns
int Updated count of books in the inventory

This function provides a menu interface for inventory management including:

  • Looking up books
  • Adding new books
  • Editing existing books
  • Deleting books
  • Returning to the main menu

The function handles user input validation and calls appropriate functions for each inventory operation. It continues to display the menu until the user chooses to return to the main menu.
