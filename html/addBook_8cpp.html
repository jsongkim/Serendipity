Returns
Updated count of books in inventory after addition

This function provides an interactive menu for:

  • Entering all book details (title, ISBN, author, etc.)
  • Validating numeric inputs
  • Saving the book to inventory when complete
  • Returning to previous menu

The function maintains a temporary copy of all fields until the user chooses to save the book to the database.
