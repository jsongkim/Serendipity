This function allows users to search for books by title (case-\/insensitive) with partial matching capability. It skips out-\/of-\/stock books (quantity \texorpdfstring{$<$}{<}= 0) and prompts the user to confirm before returning a found book\textquotesingle{}s index.

The search process\+:
\begin{DoxyEnumerate}
\item Prompts user for a title to search
\item Performs case-\/insensitive partial matching
\item Displays matches with available quantities
\item Asks for confirmation before returning a match
\item Handles various edge cases (empty input, no matches, user cancellation) 
\end{DoxyEnumerate}
