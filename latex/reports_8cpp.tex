Displays and manages the reports menu. 

This function provides a menu interface for generating various inventory reports including\+:
\begin{DoxyItemize}
\item Inventory listing
\item Wholesale value report
\item Retail value report
\item Quantity-\/based listing
\item Cost-\/based listing
\item Age-\/based listing
\end{DoxyItemize}

The function handles user input validation and continues to display the menu until the user chooses to return to the main menu. Each menu option currently displays a placeholder message indicating the selected option.

The menu features\+:
\begin{DoxyItemize}
\item ASCII-\/art border for visual appeal
\item Input validation for menu choices
\item Proper screen clearing between menu displays
\item Consistent user feedback for selections 
\end{DoxyItemize}
